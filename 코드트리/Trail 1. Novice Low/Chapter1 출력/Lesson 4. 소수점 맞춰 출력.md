**소수점 맞춰 출력하기**
실수값을 갖는 변수의 경우 소수점 자리를 맞춰 출력하는 경우가 대다수이다. 이 경우에는 먼저 다음과 같이 %를 사용해 볼 수 있다. 다음은 소수점 4째자리까지 값을 반올림하는 출력 코드이다.
*방법1*
```python
a = 33.567268
print("%.4f" % a)
```
출력 결과
```python
33.5673
```
format함수를 이용하는 것도 가능하다. 기존 포맷이 {0}이었다면 그 뒤에 : 를 붙여 포맷을 지정해 주면 된다.
*방법2*
```python
a = 33.567268
print("{0:.4f}".format(a))
```
출력 결과
```python
33.5673
```

*방법3*
```python
a = 33.567268
print(f"{a:.4f}")
```
출력 결과
```python
33.5673
```
