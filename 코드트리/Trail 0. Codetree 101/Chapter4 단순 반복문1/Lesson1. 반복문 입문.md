
### C

for 문이란?
	-> for 문은 주어진 작업을 원하는 만큼 반복할 때 사용하는 구문이다.

- for문의 기본 구조
```c
for(초기화; 조건 검사; 증감){
    실행
}
```
예
```c
for(i = 0; i < 10; i++){
    printf("%d\n", i);
}
```

### C++
- for문의 기본 구조
```c
int i;
for(i = 0; i < 반복횟수; i++){
    실행할 코드
}
```
예
```c
int i;
for(i = 0; i < 100; i++){
    cout << "안녕하세요\n";
}
// 출력:
// 안녕하세요
// 안녕하세요
// 안녕하세요
// ... (100회 반복)
// 안녕하세요
```

### Python
- for문의 기본 구조
```c
for i in range(반복횟수):
    실행할 코드
```
예
```c
for i in range(100):
    print("안녕하세요")
# 출력:
# 안녕하세요
# 안녕하세요
# 안녕하세요
# ... (100회 반복)
# 안녕하세요
```
