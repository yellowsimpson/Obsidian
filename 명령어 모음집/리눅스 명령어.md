**$pwd**
**→ 현재 디렉터리 확인**
**$ls**
**→(list)의 줄임말, 현재 디렉터리에 있는 파일 확인**
**$clear**
**→ 화면 비우기**
**$cd**
**→(change directory)의 줄임말, 디렉토리로 이동**
**$cd ..**
**→상위 디렉터리로 이동**
**$cd ~**
**→홈 디렉터리로 이동**
**$cp (복사할 파일 이름) (새로 복사할 파일 이름)**
**-> 파일 디렉터리 복사**
**$mv (이동 시킬 파일이름) (이동 시킬 위치)
-> 파일 디렉터리 이동**
**$mkdir 파일이름**
**→(make directory)의 줄임말, 하위 디렉터리를 만들 때**
**$rm -rf 파일이름** 
**→(remove)의 줄임말, 하위 디렉터리 삭제**
*파일 지울 땐 항상 주의할 것!!!*
