
- Windows에서 samba 
[방법1]
1. 파일 탐색기에서 **"내 PC"** > **"네트워크 드라이브 연결"**
2. 폴더에 `\\라즈베리파이_IP\Shared` 입력
3. 사용자 이름/암호 입력    
4. 원하는 드라이브 문자 선택 후 완료

[방법2]
1. 주소 창에  __//라즈베리파이 주소/호스트 이름__ 치고 이름이랑 주소 치면 들어가짐
